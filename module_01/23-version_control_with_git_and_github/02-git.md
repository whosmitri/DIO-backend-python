# Git

* Sistema de controle de versão distribuído, amplamente utilizado no mercado atual.
* Software gratuito e open source.
* Permite criar ramificações (branching) e realizar fusões (merging) de forma eficiente.
* Leve, rápido e ideal para trabalho colaborativo (especialmente em times e projetos remotos).

## Fluxo Básico do Git
* `git clone`: cria uma cópia local de um repositório remoto.
* `git commit`: registra alterações realizadas no repositório local, salvando o histórico.
* `git pull`: traz atualizações do repositório remoto e mescla com o código local.
* `git push`: envia alterações do repositório local para o repositório remoto.
